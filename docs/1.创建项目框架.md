### 第一步：创建根目录并初始化 pnpm 工作空间

1.  **创建并进入项目根目录**：

    ```bash
    mkdir CoBoard
    cd CoBoard
    ```

2.  **初始化根目录 `package.json`**：

    ```bash
    pnpm init
    ```

3.  **创建 `pnpm-workspace.yaml` 配置文件**：
    在根目录下创建一个名为 `pnpm-workspace.yaml` 的文件，内容如下：
    ```yaml
    packages:
      - "client"
      - "server"
    ```
    _这告诉 pnpm，我们的项目包含两个子包：前端的 `client` 和后端的 `server`。_

---

### 第二步：初始化前端项目 (client)

我们将在根目录下直接创建 Vue 3 客户端：

1.  **使用 Vite 创建 Vue 3 项目**：
    在根目录下运行以下命令（它会自动在 `client` 目录下生成代码）：

    ```bash
    pnpm create vite client --template vue
    ```

    _(如果提示需要安装 create-vite，输入 `y` 确认即可。)_

2.  **安装前端基础依赖**：
    进入 `client` 目录并安装依赖（如 Vue Router, Pinia）：
    ```bash
    cd client
    pnpm install vue-router@4 pinia
    cd ..
    ```

---

### 第三步：初始化后端项目 (server)

1.  **创建并进入 `server` 目录**：

    ```bash
    mkdir server
    cd server
    pnpm init
    ```

2.  **安装后端运行依赖**：
    安装 Express 及其核心中间件：

    ```bash
    pnpm add express cors dotenv mysql2 @prisma/client
    ```

3.  **安装开发依赖**：
    安装 Prisma 相关的开发工具和代码热更新工具：

    ```bash
    pnpm add prisma nodemon -D
    ```

4.  **初始化 Prisma 数据库配置**：
    直接在 `server` 目录下初始化 Prisma（指定 MySQL）：

    ```bash
    npx prisma init --datasource-provider mysql
    ```

    _此时 `server` 目录下会生成 `prisma/schema.prisma` 文件和 `.env` 环境变量文件。_

5.  **退回项目根目录**：
    ```bash
    cd ..
    ```
